# GameDev

## Steuerung
<kbd>A</kbd> <kbd>D</kbd> : Laufen  
<kbd>Shift</kbd> gedrückt halten : Sprinten  
<kbd>Left Mouse</kbd> : Schießen  
<kbd>ESC</kbd> oder <kbd>I</kbd> : Inventar öffnen/schließen
